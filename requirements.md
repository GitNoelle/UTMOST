# Requirements

The software is developed and tested in Linux and Mac OS environments. 

* Python 2.7 

* numpy (>=1.11.1)

* scipy (>=0.18.1)

* pandas (>=0.18.1)

* rpy2 (==2.8.6)

* R is needed for GBJ testing.

* GBJ ([0.5.0](https://cran.r-project.org/web/packages/GBJ/index.html))
